# SWR Prettier config

## Installation

1. Install npm package

```shell
npm install @swrlab/swr-prettier-config
```

1. Include in `package.json`:

```json
{
  "prettier": "@swrlab/swr-prettier-config"
}
```

2. Or create a `.prettierrc.json` file with a link:

```json
"@swrlab/swr-prettier-config"
```
